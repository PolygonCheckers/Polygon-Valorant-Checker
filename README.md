**Status:** Archived – Polygon has shut down. No longer operational.  
Open-sourced for archival purposes. No future updates. Credit Polygon if used.

## What is Valorant Checker - Polygon?
Polygon's Valorant account checker. Validates Riot accounts, captures balance, full inventory, rank, ban status via PyQt5 GUI.

## Features
- Validates Valorant accounts
- Captures VP, Radianite, Kingdom Credits
- Full inventory (skins, buddies, cards, titles)
- Detects banned/restricted accounts
- Shows current/peak rank, level
- Captures creation date, country, region
- Email/phone verification status
- Discord webhook support
- hCaptcha solver integration
- Modern graphical interface

## How to Install
1. Install Python 3.9–3.11
2. `git clone` the repo
3. `cd` into the folder
4. `pip install -r requirements.txt`
5. Run: `python new.py`

Dependencies are in `requirements.txt`
